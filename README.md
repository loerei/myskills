# MySkills

A curated compilation of AI agent skills I use during my work. This repository contains both self-created skills and those aggregated from various external sources (such as Matt Pocock).

## Usage

To load these skills into your agentic coding workspace, clone this repository directly into the `.agents/skills` directory:

```bash
git clone https://github.com/loerei/myskills.git .agents/skills
```
